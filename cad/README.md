# CAD files — OpenBikeRadar enclosure

3D print files will be added here during Day 12 of the build series.

## Planned enclosure

- Fits: ESP32 DevKit + LD2410 + OLED 0.96" + 18650 battery
- Opening for sensor (pointing backward, 20–30° tilt)
- Opening for OLED screen
- Hole for USB-C charging cable
- Slot for on/off switch
- Mounts to standard 31.6mm seatpost (with strap or bolt clamp)

## Plan B (if 3D printing is delayed)

Use a small plastic project box (e.g. Hammond 1551 series).
Cut openings with a hobby knife.
Mount sensor with hot glue inside.
